POST /analyze

Analyzes a URL for phishing detection.

    "url": "https://example.com",
    "detailed_analysis": true
}
Response:

{
    "prediction": "safe",
    "confidence": 0.98,
    "features": {
        "url_length": 21,
        "special_chars": 0,
        "suspicious_keywords": false
    }
}
