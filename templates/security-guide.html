Security Guide: Protect Yourself from Phishing

Common Phishing Indicators

Suspicious URLs
  • • Misspelled domain names (e.g., 'g00gle' instead of 'google')
  • • Unusual TLDs (.tk, .ml, etc.)
  • • IP addresses instead of domain names
  • • Extremely long URLs with random characters
Content Red Flags
  • • Urgent action required messages
  • • Requests for sensitive information
  • • Poor grammar or spelling
  • • Offers that seem too good to be true

Best Practices

Verify the Source

Always check the sender's email address or website URL carefully. Don't trust display names alone.

Use Protection

Keep your software updated, use antivirus protection, and enable two-factor authentication when possible.

Be Cautious

Never share sensitive information through email or unexpected pop-ups. Legitimate services won't ask for this.

If You Suspect You've Been Phished

Take These Steps Immediately:
  1. Change passwords for any potentially compromised accounts
  2. Contact your bank if financial information was involved
  3. Report the incident to relevant authorities
  4. Monitor your accounts for suspicious activity
  5. Use our URL checker tool to verify suspicious links
